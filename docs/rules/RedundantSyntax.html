RedundantSyntax

This rule removes redundant syntax. Currently the only syntax it removes is the final keyword on an object.

Example:

- final object foo
+ object Foo

Note: in Scala 2.12 and earlier removing the final modifier will slightly change the resulting bytecode - see this bug ticket for further information.
